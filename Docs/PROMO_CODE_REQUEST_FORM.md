# Promo Code Request Form Copy

Use this if you create a Google Form for tester promo-code requests.

## Form title

`Relaydex closed test promo code request`

## Form description

```text
Use this form after you joined the Relaydex Android tester group and opted in on the Google Play closed-test page.

Google Play may still show the paid price before code redemption.
Approved testers will receive one promo code for free access to the closed test.

Please submit this form only once.
```

## Suggested fields

1. Email address used for Google Play opt-in
2. Joined the Google Group?
3. Completed Play opt-in?
4. Android device model
5. Country or region
6. Optional notes

## Confirmation message

```text
Request received.

If your request looks valid, one promo code will be sent through the recruitment channel you were asked to use.
Please redeem the code in Google Play, install the app, and stay opted in for at least 14 days.
```
