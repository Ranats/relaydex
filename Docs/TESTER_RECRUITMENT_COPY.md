# Tester Recruitment Copy

Use this when recruiting Android closed-test users.

## Short GitHub / README block

```text
Android closed testers wanted for Relaydex.

Relaydex is an independent fork of Remodex focused on controlling local Codex running on Windows from Android.

If you want to help test:
1. Join the tester group
2. Open the Play opt-in link
3. Request one promo code
4. Redeem the code and install the app
5. Stay opted in for 14 days

This is not the official Remodex app.
Google Play may show the paid price before redemption.
Closed testers can receive one free promo code after joining the tester group and opting in.

Join group: https://groups.google.com/g/relaydex-android-testers
Opt in: https://play.google.com/apps/testing/io.relaydex.android
Install: https://play.google.com/store/apps/details?id=io.relaydex.android
```

## Landing page template

```text
Relaydex Android closed test

What this is:
Relaydex is an independent fork of Remodex. It lets you control local Codex running on your own Windows PC from Android.

How to join:
1. Join the tester group:
https://groups.google.com/g/relaydex-android-testers

2. Opt in on Google Play:
https://play.google.com/apps/testing/io.relaydex.android

3. Request one promo code for the paid app.

4. Redeem the code and install the app from Google Play:
https://play.google.com/store/apps/details?id=io.relaydex.android

5. Keep the app installed and remain opted in for at least 14 days.

Feedback:
https://github.com/Ranats/relaydex/issues

Important:
- This is not the official Remodex app.
- You need an Android device and a Google account.
- The app is a remote control client; Codex still runs on your own computer.
- Google Play may show the paid price before redemption.
```
